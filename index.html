Database Visualization Project
This was a partner project for my databases class. In this project, we had to take a dataset and created a database out of it and then create visualizations off that database. We used all the Netflix movies and shows as our dataset. We designed our schemas through normalization and ER and RM diagrams. Once we designed our database, we wrote a python script to read the CSV file and insert the data into a Google Cloud MySQL database. We then used matplotlib to create visualizations on our data.
